# pod042

Primary laptop. Hybrid Intel + NVIDIA GPU with PRIME offload.

## Files

| File | Purpose |
|---|---|
| `configuration.nix` | Imports `core`, `audio`, `boot`, `network`, `desktop`, plus host-local files |
| `host.nix` | `networking.hostName = "pod042"` + `local.host.displays` ([[Modules/Host options]]) |
| `hardware.nix` | Imports `modules/hardware/{bluetooth,graphics,nvidia,power-profiles,touchpad}.nix` + sets `hardware.nvidia.prime.{intel,nvidia}BusId` |
| `hardware-configuration.nix` | Auto-generated; do not edit |
| `users.nix` | Imports `users/{neburion,nululy}` |

## What's enabled

System (via `modules/`):
- Core: nix flakes + gc, locale `Asia/Tokyo` / `en_US.UTF-8`, sudo-no-password, fish default shell + `EDITOR=nvim`
- Boot: GRUB EFI with os-prober, 10 generations kept
- Audio: PipeWire (alsa + pulse + wireplumber)
- Network: NetworkManager, OpenSSH (key-only, no root), localsend
- Desktop: Hyprland + SDDM, wayland env vars, fonts, gaming (steam + jdk25), rustdesk, kdeconnect, flatpak, dconf, brightnessctl, xdg-user-dirs, electron-38 allowlist
- Hardware: bluetooth + blueman, graphics, NVIDIA (with PRIME offload — bus IDs `PCI:0:2:0` / `PCI:1:0:0`), power-profiles-daemon, libinput

Users (per [[Users/Overview]]):
- `neburion` (wheel) — full home stack incl. dev + nvf + gaming
- `nululy` (wheel) — desktop + cli + dev, no nvf

## Display configuration

Set in `host.nix`:
- Primary 1920×1080 (used for SDDM)
- `monitors.builtin`: `eDP-1` @ `1920x1080@120` at `0x0`
- `monitors.external`: `HDMI-A-1` @ `1920x1080@144` at `1920x0`

These propagate to hyprland via `hostConfig` ([[Concepts/Host config (data flow)]]).
