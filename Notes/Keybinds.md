All keybinds use `SUPER` (`$mod`) as the modifier. Configured in `home-modules/desktop/wm/hyprland/keybinds.nix`.

## Apps

| Keybind | Action |
|---------|--------|
| `SUPER + Enter` | Terminal (ghostty) |
| `SUPER + Space` | App launcher (wofi drun) |
| `SUPER + SHIFT + Space` | Theme switcher |
| `SUPER + ALT + Space` | Power menu |
| `SUPER + F` | File manager (thunar) |
| `SUPER + W` | Wallpaper manager (waypaper) |
| `SUPER + A` | Audio manager (pavucontrol) |
| `SUPER + SHIFT + Escape` | Task manager (btop in ghostty) |
| `SUPER + B` | Web browser (zen) |
| `SUPER + M` | Music player (spotify) |
| `SUPER + N` | Notes (obsidian) |
| `SUPER + D` | Discord (vesktop) |
| `SUPER + C` | Messenger (signal) |
| `SUPER + G` | Heroic launcher |
| `SUPER + SHIFT + G` | Steam |
| `SUPER + P` | KeePassXC |
| `SUPER + S` | LocalSend |

## Windows

| Keybind | Action |
|---------|--------|
| `SUPER + Backspace` | Close window |
| `SUPER + T` | Toggle float |
| `SUPER + SHIFT + S` | Toggle split |
| `SUPER + H/L/K/J` | Move focus left/right/up/down |
| `SUPER + SHIFT + H/L/K/J` | Move window left/right/up/down |

## Workspaces

| Keybind | Action |
|---------|--------|
| `SUPER + 1–0` | Switch to workspace 1–10 |
| `SUPER + SHIFT + 1–0` | Move window to workspace 1–10 |

## Session

| Keybind | Action |
|---------|--------|
| `SUPER + Escape` | Exit Hyprland |

## Screenshots

| Keybind | Action |
|---------|--------|
| `Print` | Region screenshot (slurp) → saved + clipboard |
| `SHIFT + Print` | Full screenshot → saved + clipboard |

Screenshots saved to `~/Media/Image/Screenshot/`.

## Audio

| Keybind | Action |
|---------|--------|
| `SUPER + =` | Volume +5% |
| `SUPER + -` | Volume -5% |
| `XF86AudioRaiseVolume` | Volume +5% (capped at 100%) |
| `XF86AudioLowerVolume` | Volume -5% |
| `XF86AudioMute` | Toggle mute |
| `XF86AudioMicMute` | Toggle mic mute |
| `XF86MonBrightnessUp` | Brightness +10% |
| `XF86MonBrightnessDown` | Brightness -10% |

## Mouse

| Bind | Action |
|------|--------|
| `SUPER + LMB drag` | Move window |
| `SUPER + RMB drag` | Resize window |
